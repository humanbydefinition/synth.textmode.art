# Contributing to synth.textmode.art

> [!IMPORTANT]
> **Work-in-progress**: This project is currently in active development. APIs, file structures, and features may change rapidly. Building on top of this project is encouraged, but please be aware of likely breaking changes.

Thank you for your interest in contributing to `synth.textmode.art`!

## Getting started

Please refer to the [README](README.md) for instructions on how to set up the development environment locally.

1.  **Fork the repository** on GitHub.
2.  **Clone your fork** locally.
3.  **Install dependencies** with `npm install`.
4.  **Start the dev server** with `npm run dev`.

## Code style

This project uses **Prettier** for code formatting and **ESLint** for linting.

-   **Format code**: Run `npm run format` to automatically format your code.
-   **Lint code**: Run `npm run lint` to check for issues.

Please ensure your code is formatted and lint-free before submitting a Pull Request.

## Pull assignments and issues

-   **Issues**: If you find a bug or have a feature request, please [open an issue](https://github.com/humanbydefinition/synth.textmode.art/issues).
-   **Pull Requests**:
    1.  Create a new branch for your feature or fix.
    2.  Make your changes.
    3.  Run `npm run build` to verify that the project builds correctly.
    4.  Submit a Pull Request providing a clear description of your changes.
